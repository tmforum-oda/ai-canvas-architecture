# AI-Native ODA Canvas Architecture

This repository documents the architecture of the **AI-Native ODA Canvas**.

The AI-Native Canvas extends the TM Forum **Open Digital Architecture (ODA) Canvas** to support **agentic and AI-enabled workloads**, providing a standardized execution environment for applications interacting with models, tools, data platforms, and enterprise systems.

The architecture defines how **agentic components declare and access resources** such as:

- LLM models
- other agents (A2A interfaces)
- tools (MCP interfaces)
- APIs
- data platforms


using **declarative metadata and Kubernetes operators**.

---

# Purpose of this Repository

This repository serves as the **source of truth for the AI-Native Canvas architecture**.

It provides:

- Architecture Decision Records (ADRs)
- Architecture diagrams
- Architecture definition documentation
- Working group discussions
- Open architecture questions

---

# Architecture Vision

AI-Native ODA Canvas provides a **portable, multi-vendor execution environment for agentic applications**.
